# Homebrew Tap for fackr

This is the Homebrew tap for [fackr](https://github.com/TenseleyFlow/fackr), a terminal text editor written in Rust.

## Installation

```bash
brew tap TenseleyFlow/fackr
brew install fackr
```

## Usage

```bash
fackr [file]
```

## About

fackr is a Rust reimplementation of [facsimile](https://github.com/FortranGoingOnForty/facsimile), providing a modern terminal text editor with VSCode-style keybindings.
